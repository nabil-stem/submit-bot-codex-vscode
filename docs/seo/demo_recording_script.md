# Demo Recording Script (60-90 seconds)

Use this script to record a clean demo video for GitHub and store listings.

## Scene 1 (10s): Safety First

- Open config/options.
- Show allowlist only.
- Show dry-run enabled.

## Scene 2 (20s): VS Code Desktop Mode

- Open Codex submit prompt.
- Show tray status.
- Trigger a dry-run match in logs.
- Switch to live mode and show one controlled click.

## Scene 3 (20s): Browser Extension Mode

- Open allowed URL.
- Show popup global toggle + per-site toggle.
- Show in-page active overlay and last action time.

## Scene 4 (10s): Guardrails

- Open a non-allowlisted URL.
- Show no clicks happen.

## End Card (10s)

- Repo URL
- Extension store link
- "Start in dry-run mode"
