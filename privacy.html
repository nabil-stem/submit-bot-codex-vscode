Privacy Policy

Last updated: February 23, 2026

This Privacy Policy applies to Submit Auto-Clicker (desktop app and browser extension).

Data Collection

Submit Auto-Clicker does not send user data to external servers by default.

The browser extension stores configuration locally in browser storage:

  • URL allowlist patterns
  • button text patterns
  • container selector and timing settings
  • global/per-site and dry-run toggles
  • local last-action timestamps

The desktop app stores local configuration and local logs on the user's machine.

Data Sharing

The project does not include built-in third-party analytics, ad trackers, or remote data APIs for core behavior.

Data Use

Local data is used only to operate automation rules selected by the user and to provide status information.

  • Data is not sold.
  • Data is not used for advertising.
  • Data is not used for creditworthiness or lending decisions.

User Controls

  • Automation can be disabled at any time.
  • Extension local storage can be cleared from browser settings.
  • Desktop config/log files can be deleted locally.

Security Responsibility

Automation can trigger real actions. Users are responsible for safe configuration, especially allowlists and live-click mode.

Contact

For support, open an issue at GitHub Issues.
